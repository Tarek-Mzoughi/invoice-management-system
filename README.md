# Invoice Management System

A full-stack web application designed to manage invoices, quotations, delivery notes, customer orders, payments, products, activities, taxes, users, roles, permissions, dashboards, and company configurations.

This project was developed as part of a final-year academic project and focuses on building a complete, scalable, and professional business management platform.

---

## Project Overview

The platform provides a centralized solution for managing the full commercial document lifecycle, from quotation creation to invoice generation, delivery tracking, payment management, and administrative monitoring.

The system includes a modern web interface, a structured backend API, role-based access control, document sequencing, PDF generation, dashboards, and multi-company configuration.

---

## Main Features

- User authentication and authorization
- Role-based access control
- Company and activity management
- Customer and supplier management
- Product and price list management
- Quotation management
- Customer order management
- Delivery note management
- Goods issue note management
- Invoice management
- Credit note and return note management
- Payment tracking
- Tax and fiscal configuration
- Document sequence management
- PDF document generation
- Dashboard and statistics
- Activity logs
- Multi-language support
- Administrative configuration interface
